## Notes

- Each page is standalone and keeps the failed branch before the successful
  path.
- Exact commands, payloads, and printed outputs are included wherever they were
  preserved from the solve process.
- Raw Codex transcript archives were kept local instead of being pushed
  publicly because they contain browser cookies, tokens, and other sensitive
  competition-session material.
